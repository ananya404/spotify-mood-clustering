# 🎵 Spotify Music Mood Clustering

Unsupervised ML project that clusters 232,725 Spotify songs 
into 4 mood groups using KMeans clustering.

## Mood Clusters
- 🎉 Party / Happy
- 😢 Sad / Acoustic
- 🎸 Energetic / Hype
- 😌 Chill / Neutral

## Tech Stack
- Python, Scikit-learn, KMeans
- PCA & t-SNE visualization
- Streamlit deployment

## How to Run
pip install streamlit scikit-learn pandas matplotlib
streamlit run app.py
